## Usage

Select the theme and you are ready to go!

## Installation

1.  Open Extensions sidebar panel in VS Code. View → Extensions
1.  Search for `Raijū`
1.  Click Install
1.  Click Reload
1.  File > Preferences > Color Theme > Raijū
